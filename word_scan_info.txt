- If your flagged phrases contain characters like hyphens, parentheses, or periods, you must run them through this function before building your search pattern so Python treats them as literal text instead of regex commands.

Handling UI Output:
Because re.finditer() provides index ranges, you can easily use those character positions to slice the string later.
