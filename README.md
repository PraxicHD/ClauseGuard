# ClauseGuard

A TOS scanning program that scans pasted Terms of Service information and flags buzz words, categorizing them, and printing it to the user for further revi

#Features (IN PROGRESS)
Text scanning
Flags words/phrases that you should pay close attention to
Provides a risk score 0 - 100

#Future Plans
File scanning
Export the report to a file
User added flags

#CATEGORIES
Money (automatic renewal, cancellation fee, no refunds, price may change)
Legal Rights (arbitration, class action waiver, waive rights)
Privacy (share data, third parties, tracking, sell personal information)
Account Control (suspend account, terminate access, remove content)
Contract Changes (may update terms, continued use means acceptance)
Liability (not responsible, limited liability, at your own risk)
